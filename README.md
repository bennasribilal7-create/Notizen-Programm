# Notizen-Manager
Funktionen: Neue Notizen speichern  Alle vorhandenen Notizen anzeigen  Einzelne Notizen löschen  Alle Notizen löschen  Benutzerfreundliche Konsolensteuerung mit Auswahlmenü  Endlosschleife für mehrere Operationen, bis das Programm beendet wird
Gespeichert wird automatisch in einer notizen.json.

Bedienung

1 = neue Notiz
2 = alle anzeigen
3 = eine löschen
4 = alles löschen
5 = beenden
